About
---------------

Circle webhost is a basic website hosting server written in c#. It has a variety of configurations for you website. All the basic foundations for a webserver have been written
and its up to you what you want to implement or change.

How to Install (Build from source)

Dependances
---------------

.NET Core SDK / .NET Core Runtime
https://github.com/dotnet/core/blob/master/release-notes/download-archives/2.0.6-download.md

How to Build / Install
-------------------------

In a CMD or Terminal window, within the working directory of the project ( ./CircleWebHost/ ):

dotnet run

then you are good to go!

How to Configure
--------------------

In the ./CircleWebHost/bin/Debug/netcoreapp2.0/config folder there exists three config files.
The first config file is not implemented yet, but DO NOT DELETE IT. The second is the webserver
config file for setting the directory of the website and other settings. I recommend to not change
it unless necessary. lastily, there is the virtual disk service vonfig file. That file is for masking
the filestructure of the webserver, when you use it, it is for mapping a fake URL to a real, physical
one. When using the virtual directory service, the website HTML files and associated hyperlinks must
refer to the virtual URL written in that file.

Please follow the convention of the associated config files, or rist corrputing the program.

Deficiencies
------------------------
The server is currently hard-coded to make a simple website. The server does not save any website sessions
to the hard drive. Anything saved into it will be deleted upon closing the server. Feel free to modify this
code if you wish.
